# Elastic Data Pusher

This program is desigend to push data to an elasticsearch instance.

## Setup

### Tested with

- Python >= 3.7.x

### **Setup virtual python development environment**

```bash
$ git clone <repo>
$ cd elasticDataPush
$ python3 -m venv venv
$ source venv/bin/activate
(venv) $ pip install -r requirements
```
